# HR Analytics Dashboard

## 📖 Project Description

The HR Analytics Dashboard is an interactive Power BI project developed to analyze employee data and provide meaningful HR insights. This dashboard helps organizations monitor employee distribution, salary trends, employee satisfaction, performance scores, and gender distribution. Interactive filters allow users to explore data by department, making it easier to support HR decision-making.

---

## 🎯 Objectives

- Analyze the total number of employees.
- Monitor average employee salary.
- Measure employee satisfaction and performance.
- Compare employee count across different departments.
- Analyze gender distribution.
- Provide interactive filtering using a department slicer.

## 🚀 How to Use

1. Download or clone this repository.
2. Open **HR_Analytics_Dashboard.pbix** using Power BI Desktop.
3. If prompted, refresh the dataset.
4. Use the **Department** slicer to filter employee data.
5. Explore KPIs and charts to gain HR insights.
