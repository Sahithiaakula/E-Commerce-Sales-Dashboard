# E-Commerce Sales Dashboard

## Project Overview
This project analyzes E-Commerce sales data using SQL and Power BI. The data was cleaned in Excel, imported into MySQL, analyzed using SQL queries, and visualized in an interactive Power BI dashboard.

## Tools Used
- Excel
- MySQL
- SQL
- Power BI

## Dataset
Sample Superstore Dataset

## SQL Analysis
Performed 30 SQL queries including:
- Total Sales
- Total Profit
- Total Orders
- Total Quantity
- Sales by Category
- Profit by Region
- Top Customers
- Top Products
- Sales by Segment
- Sales by State
- Monthly Sales Trend
- And more...

## Dashboard KPIs
- Total Sales
- Total Profit
- Total Quantity Sold
- Total Orders

## Dashboard Visuals
- Monthly Sales Trend
- Profit by Region
- Sales by State
- Sales by Category
- Sales by Segment
- Top 10 Customers
- Top 10 Products

## Project Workflow
1. Cleaned the dataset in Excel.
2. Imported the CSV into MySQL.
3. Wrote 30 SQL queries for business insights.
4. Connected MySQL with Power BI.
5. Built an interactive dashboard.
